# 시간 속의 숫자 찾기

------------
## 사용 라이브러리/클래스/함수
java.io.*   
java.util.*   
BufferedReader   
InputStreamReader
------------
## 문제
문제 설명   
몇 개의 3이란 숫자가 나올까?   

00시 00분 00초부터 N시 59분 59초까지의 시각 중에서   
3이 포함된 숫자의 개수 출력 (0<=N<=23)   

입력 : 5   
출력 : 11475


## 풀이

------------ 
시간을 입력을 받고 반복문을 통해    
00시 00분 00초부터 h시 59분 59초까지 비교를 해준다.   
time.contains("3"); 해당 함수로 문자열 안에 3이 있다면 count 를 증가 시켜 준다.
